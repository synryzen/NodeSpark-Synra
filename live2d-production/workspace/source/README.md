# Source Art

Place Synra's layered source art here.

Expected main file:

```text
synra.psd
```

Approved design references in this folder:

```text
synra-final-design-neutral.png
synra-final-design-wave.png
```

Use the neutral image as the base source-art pose. Use the wave image as the
motion/anatomy reference for rigging.

The PSD needs independent layers for face, eyes, mouth, hair, arms, hands,
dress, frills, ribbons, jewelry, and NodeSpark details. Follow:

```text
../../ART_BRIEF.md
../../layer_manifest.json
```

Flat PNG art is useful as a reference, but it is not enough for the final
Live2D rig.
