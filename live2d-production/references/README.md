# References

This folder stores local visual references for Synra Live2D production.

- `current-vrm-fallback.jpg` shows the temporary character currently running in
  the app. It is a baseline to improve from, not the final design target.
- `synra-final-design-neutral.png` is the approved base-pose design for the
  final Live2D model.
- `synra-final-design-wave.png` is the approved palm-forward wave anatomy
  reference.

Add the dress, wave, face, and outfit references here before building a final
artist/rigger handoff package.
